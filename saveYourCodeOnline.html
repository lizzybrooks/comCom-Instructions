Creating a GitHub Repository for Node Projects

Initialize your project

Navigate to your project's root directory and run:

git init

Create .gitignore file

Add node_modules/ to your .gitignore file to exclude it from Git tracking.

node_modules/

Add your project files to the repository

Add and commit your files to Git:

git add .
git commit -m "Initial commit"

Create a repository on GitHub

Create a new repository on GitHub via the website.

Link your local repository to GitHub

Link your local repository with the GitHub repository:

git remote add origin <REPOSITORY_URL>
git branch -M main
git push -u origin main

Push your local commits to the GitHub repository

git push -u origin main

Install node modules

Run npm install to install dependencies after cloning the project.

npm install
